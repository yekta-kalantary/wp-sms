# قالب پرامپت توسعه‌ی Core

```text
این مخزن monorepo اکوسیستم Yekta SMS است. ابتدا README.md و سپس docs/ را کامل بررسی کن. بعد ساختار pluginهای موجود را تحلیل کن. چیزی را حدس نزن.

درخواست:
قابلیت [نام قابلیت] را به core اضافه کن.

قواعد:
1) اول مشخص کن این قابلیت عمومی است یا نه.
2) اگر عمومی است، ابتدا docs/architecture و در صورت نیاز ADR را به‌روزرسانی کن.
3) سپس implementation، tests، migration notes و release docs را تکمیل کن.
4) backward compatibility را بررسی کن.
5) dependency جدید غیرضروری اضافه نکن.
6) در پایان گزارش نهایی را با فرمت اجباری README بده.
```
