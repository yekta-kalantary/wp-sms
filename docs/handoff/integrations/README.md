# Handoffهای Integration

فایل‌های handoff مربوط به integrationها در این مسیر قرار می‌گیرند.

## انتظار
هر handoff باید شامل این بخش‌ها باشد:
- هدف plugin
- dependencyها
- compatibility requirements
- trigger map
- settings/mapping schema
- placeholderها
- idempotency strategy
- admin tools
- test cases
- edge cases
- release checklist
