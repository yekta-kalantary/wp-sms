# Handoffهای Gateway

فایل‌های handoff مربوط به gatewayها در این مسیر قرار می‌گیرند.

## انتظار
هر handoff باید شامل این بخش‌ها باشد:
- هدف plugin
- provider boundary
- dependencyها
- file map
- settings schema
- health check
- request/response normalization
- error mapping
- logging redaction
- test cases
- release checklist
