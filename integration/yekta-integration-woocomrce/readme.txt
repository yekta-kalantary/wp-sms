WooCommerce integration plugin for Yekta SMS Core.

== Description ==

This plugin maps WooCommerce order events to Yekta SMS Core dispatch requests.

== Dependencies ==

* Yekta SMS Core
* WooCommerce

== Features ==

* Dependency checks and graceful degradation.
* HPOS and checkout blocks compatibility declarations.
* Event mapping settings and manual resend.
* Recipient resolution and placeholder rendering.
* Idempotency guard with order meta markers.
